Academic Projects

Predicting Author Demographics from textual data Aug 2020 - Dec 2020

George Mason University

  • Developed a model that predicts the age and gender of an author given textual data using classification and regression algorithms such as Multivariate Linear Regression with Lasso Regularization, Support Vector Machines and Deep Neural Networks
  • Stylistic (POS-Tags, Unigrams), content-based (Special Words, Slang Words) and word embeddings (Word2Vec) were used to extract features from the text, that prove as important descriptors towards our prediction
  • Neural Networks with one hidden layer with the feature set extracted using Word2Vec gave the best MAE of 4.10 and a r-coefficient of 0.64 for age prediction and an accuracy of 89% for gender prediction
  • Technologies Used: Python, NLTK, Scikit-learn, Keras, Matplotlib, Gensim

Hybrid Movie Recommendation System Nov 2020 - Dec 2020

George Mason University

  • Developed a Hybrid Recommender System using Alternating Least Squares and Item-Item Collaborative Filtering in PySpark using SparkML on the Movie Lens 20M dataset
  • The data was stored and read from an Amazon S3 Bucket and our PySpark notebook was deployed using an Amazon EMR Cluster to distribute the data and processing across a resizable cluster of EC2 instances
  • The hybrid recommender system gave us an RMSE of 0.69 which is an improvement over RMSE calculated on the two recommendation algorithms individually, which was 0.86 and 0.79, respectively
  • Technologies Used: Python, Spark, SparkML, AWS EC2, AWS EMR

Topic Modelling using LDA Sep 2020

George Mason University

  • Topic Modelling is an unsupervised classification algorithm for topic modelling to group documents based on a given set of important terms/keywords.
  • We make the use of pyspark.ml.clustering.LDA for performing LDA on our document corpus.
  • For finding the value of k, after multiple runs, the number of topics being 25 was the perfect fit and gave the best results. The value of logPerplexity for k = 25 is 7.712397571620074, and this was the a good balance between number of topics and the perplexity that we could find for a set of k values between 10 and 250.
  • Technologies Used: Python, Spark, SparkML

Sentiment Analysis on the Sentiment140 Dataset Jun 2020

George Mason University

  • Performed Sentiment Analysis on the Sentiment140 Dataset. It contains 1,600,000 tweets extracted using the twitter api. The tweets have been annotated (0 = negative, 4 = positive) and they can be used to detect sentiment.
  • Using Pre-trained Glove Embeddings on the tweets. GloVe is an unsupervised learning algorithm for obtaining vector representations for words. Training is performed on aggregated global word-word co-occurrence statistics from a corpus, and the resulting representations showcase interesting linear substructures of the word vector space.
  • LSTM Long Short Term Memory are a special kind of RNN, capable of learning long-term dependencies, are used for sentiment analysis.
  • Technologies Used: Python, Matplotlib, Pandas, scikit-learn, Keras, NLTK, Gensim

YOLO Object Detection on PASCAL VOC Jun 2020

George Mason University

  • Implemented a multi-label image classifier and a YOLO-like object detector on the PASCAL VOC 2007 dataset using PyTorch and a Deep Learning VM instance from Google Cloud Platform
  • Designed and trained a deep convolutional network to predict a binary present/absent image-level label for each of the 20 PASCAL classes and developed a network structure inspired by DetNet for object detection .
  • Technologies Used: Python, GCP, PyTorch
